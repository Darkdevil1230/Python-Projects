**Phone Book Management System (Python)**

**Description**

A menu-driven Phone Book Management System built using Python.
The project uses dictionaries for in-memory state management and file handling for persistent storage.


**Features**

* Add new contacts
* Delete existing contacts
* Search contacts by name
* Display all contacts sorted alphabetically
* Persistent storage using text files
* Handles empty and missing files gracefully


**Concepts Used**

* Dictionaries
* File handling
* Functions \& modular design
* Sorting \& searching
* Exception handling


**How to Run**

*python phonebook.py*
